# 규칙
* Baekjoon/ 폴더에는 https://www.acmicpc.net/ 에서 푼 문제들만 다룬다.

    - Baekjoon/ 폴더 구조는 다음과 같다.
    - Baekjoon/ 밑에 "문제번호" 폴더를 만들고 "이름" 폴더를 만들어 소스파일을 올린다. ex) Baekjoon/1000/seunguklee/1000.cpp
```
└── Baekjoon
    ├── 1000
    │   ├── jinhyunly
    │   │   ├── 1000.cpp
    │   │   └── Main.java
    │   └── seunguklee
    │       ├── 1000.cpp
    │       ├── 1000_1.cpp
    │       └── README.md
    └── 2000
```
